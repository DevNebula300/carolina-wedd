# Carolina & Reda — Wedding Celebration

Full-screen, scroll-down wedding website built with **Vite**, **React**, and **TypeScript**.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Content & links

- **Images**: Hero and invitation use `Image 1.jpg` from `public/images`. Your `images` and `pinterest` folders are copied into `public/` so the site can load them. To use Image 2 as the hero, set `HERO_IMAGE` in `src/App.tsx` to `/images/Image%202.jpg`.
- **Placeholders** (replace in `src/App.tsx` when you have them):
  - `MAP_LINK` — Google Maps link for Jnane Rumi, Marrakech
  - `RSVP_LINK` — RSVP form or page
  - `ADD_TO_CALENDAR_LINK` — Add to calendar link
  - `ACCOMMODATIONS[].url` — Each accommodation’s website
  - `DISCOVER_SPOTS[].url` — Café Bacha, Royal Mansour, YSL Museum, Boutique X, Boutique Y
